# Solstorm Animator

## Public deploy

This app can be deployed as a static site.

Recommended:
- Vercel: import this repo and deploy from the repo root. The included `vercel.json` already points the build output at `apps/desktop/dist`.

Also supported:
- Netlify: import this repo and deploy from the repo root. The included `netlify.toml` already sets the build command and publish directory.

Local commands:

```bash
npm run build
```

After deployment, you will get a public URL you can open from any device, including iPad and phones outside your home network.
